# Copier Template Usage

## Generate interactively

```bash
copier copy ./copier_template ../my-new-lesson-site
```

## Generate from YAML answers

```bash
copier copy ./copier_template ../my-new-lesson-site --data-file ./copier_template/copier-answers.example.yml
```

## Key template features

- `authors`: YAML list of author names
- `sections`: YAML list controlling module slugs/titles/lesson slugs
- `scaffold_lessons`: auto-create section folders, section `index.rst`, and lesson `.md` files
- `theme_preset`: `colab`, `duke`, `flamin_hot`, or `custom`
- `theme_registry`: extend with additional named themes over time

The section and lesson files are generated by `.copier-helpers/scaffold_lessons.py` during template creation.
