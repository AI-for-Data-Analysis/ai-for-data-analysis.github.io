Session 1: Foundations for AI-Assisted Analytics Work
=====================================================

Date/Time: June 10, 2026, 9:00 AM-12:50 PM

Session outcome:
Participants can move from unstructured AI-assisted discovery to a reproducible, code-based analysis workflow.

Primary interface for Session 1:

- JupyterLab with Jupyter-AI for exploration and analyst-friendly iteration
- Project-local Python environment (`.venv`) for reproducibility

Required first step: `Complete setup <../setup.html>`_

Agenda:

- 9:00-9:20: Framing and workflow modes
- 9:20-9:55: Project setup (`.venv`, repository layout, run commands)
- 9:55-10:35: Prompting, planning, and context for analytics
- 10:35-10:50: Break
- 10:50-11:35: Discovery lab with a lightweight data viewer
- 11:35-11:50: Break
- 11:50-12:30: Production-mode mini analysis and checks
- 12:30-12:45: Version control and review loop
- 12:45-12:50: Wrap-up and assignment

Deliverables:

- A local project environment created and validated by the learner
- A small exploration artifact (viewer or notebook) for dataset discovery
- A reproducible analysis artifact with at least basic quality checks
- A short assumptions/limitations note
