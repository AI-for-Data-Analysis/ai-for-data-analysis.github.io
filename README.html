AI Accelerator: Analytics and Data Analysis#

Tutorial-style lessons for AI-assisted analytics from exploration to dashboard delivery.

Authors#

  • Jessica Nash, PhD - Developer and Educator, AI Specialist at Duke Innovation Co-Lab

Included sections#

  • Session 1: Foundations for AI-Assisted Analytics Work (session-1-foundations)

  • Session 2: Reusable Skills for Analytics Work (session-2-reusable-skills)

  • Session 3: From Analysis to Dashboard (session-3-analysis-to-dashboard)

  • Session 4: Dashboard Presentations and Reflection (session-4-presentations-reflection)

Getting started#

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
make html

Open _build/html/index.html.
